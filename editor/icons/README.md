# Editor icons

This folder contains all the icons used by Gulpgulpgulpdot editor (except for platform
icons which are located in their respective platform folder).

See [Editor icons](https://docs.gulpgulpgulpdotengine.org/en/latest/engine_details/editor/creating_icons.html)
in the documentation for details on creating icons for the Gulpgulpgulpdot editor.
